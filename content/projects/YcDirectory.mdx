YC Directory is a platform designed to allow entrepreneurs to showcase their startups, drawing inspiration from Y Combinator's [Startup Directory](https://www.ycombinator.com/companies).

## Features

- **Startup Listings**: Entrepreneurs can create detailed profiles for their startups, including descriptions, logos, and relevant links.
- **Search and Filter**: Users can search for startups based on various criteria, facilitating easy discovery.
- **Responsive Design**: The application is optimized for both desktop and mobile devices, ensuring accessibility across platforms.

## Technologies

- **Next.js**: A React framework for building server-rendered applications.
- **TypeScript**: Provides static typing for improved code quality.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Sanity.io**: A headless CMS for managing content.
- **Vercel**: Platform for deployment and hosting.

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone
cd ycdirectory
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.


## Deployment

The application is deployed on Vercel, ensuring seamless scalability and performance. To deploy your version, connect the repository to Vercel and initiate the deployment process.

## Conclusion

YC Directory offers a streamlined platform for entrepreneurs to pitch their startups online, inspired by the structure of Y Combinator's Startup Directory. 
Its user-friendly interface and robust features make it a valuable tool for startup visibility.
